Detailed Description

template<MIDIMessageType Type, uint8_t BankSize>
class Bankable::NoteCCKPLED< Type, BankSize >

Generic base class for classes that listen for MIDI Note, Control Change and Key Pressure events on a single address and save their value.

This version listens accross multiple banks.

Template Parameters
TypeThe type of MIDI messages to listen for:
  • MIDIMessageType::NOTE_ON
  • MIDIMessageType::CONTROL_CHANGE
  • MIDIMessageType::KEY_PRESSURE
BankSizeThe number of banks.
Examples
Bank-Button-Matrix.ino.

◆ onBankSettingChange()

void onBankSettingChange ( )
inlineoverrideprotectedvirtual

A function to be executed each time the bank setting changes.

Think of an LED that indicates whether a track is muted or not. If this LED is bankable, let's say with 4 tracks per bank, 2 banks, and a base address of 3, then this LED object keeps the state of tracks 3 and 7. When the bank setting is 0, the LED displays the state of track 3, when the bank setting is 1, the LED displays the state of track 7.
To know when to update the LED, this callback is used.
