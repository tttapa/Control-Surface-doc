Detailed Description

template<MIDIMessageType Type>
class NoteCCKPLED< Type >

Generic base class for classes that listen for MIDI Note, Control Change and Key Pressure events on a single address and turns on an LED when the value is higher than a threshold.

Template Parameters
TypeThe type of MIDI messages to listen for:
  • MIDIMessageType::NOTE_ON
  • MIDIMessageType::CONTROL_CHANGE
  • MIDIMessageType::KEY_PRESSURE
Examples
1.Note-LED.ino, 2.First-Input.ino, 6.MAX7219-NoteLED.ino, AppleMIDI.ino, Debug-MIDI-Interface.ino, Dual-MIDI-Interface.ino, and Serial-Interface.ino.
